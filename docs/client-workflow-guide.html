Non-technical operating manual

How to operate the Himaliya system

Click-by-click diagrams for signing in, managing customers, recording sales, processing orders, generating invoices and controlling payments.

Desktop: use the left sidebar · Mobile: tap the menu icon first
1

How to access the system

Admin and customer access are separate.

Admin access

  1. Open the website landing page.
  2. Choose Admin Sign In.
  3. Enter the admin email and password.
  4. The system opens the Dashboard.

Customer access

  1. Open the website landing page.
  2. Choose Order Water.
  3. Sign in, or switch to Create Account.
  4. The system opens the customer ordering portal.
If a session expires: the user is returned to the correct sign-in page. Use “Forgot password?” if access cannot be restored.
2

Add and find a customer

Create a customer manually or locate an existing portal signup.

Admin dashboard→Sidebar→Add Customer
＋

Open Add Customer

Use the customer section in the sidebar.

✍️

Enter details

Name, phone and address are required. Email is optional.

💾

Save customer

Select Add Customer. The record is stored in the cloud.

🔎

Find or edit later

Use Existing Customers or open Customer Records.

Customers created by the admin and customers who register themselves are shown together. The source label explains how each record was created.
3

Record a daily sale

Add the delivery to the correct customer’s purchase history.

Sidebar→Daily Sales Entry
🔎

Search customer

Enter the customer name or phone and choose Search Customer.

👤

Select correct record

Choose the customer card from the results.

💧

Enter sale

Choose bottle type, quantity and unit price in Record Purchase.

🧮

Check total

Confirm the calculated total amount before submitting.

✓

Save

The sale enters purchase history and updates reporting/inventory.

4

Generate and download an invoice

This is the main invoice-generation procedure.

Sidebar→Customer Records→Select customer→Filter & Export
🔎

Find customer

Search by name, phone or email, then click the customer.

📅

Select period

Choose the required purchase-history period using the filter buttons.

👁️

Review figures

Check orders, bottles, revenue and the purchase-history rows.

🧾

Choose Export PDF

The system creates an invoice number, saves the invoice and downloads the PDF.

☁️

Invoice is registered

It appears under Customer Invoices and in the Invoice Register.

Important: exporting the PDF does not mark the invoice as paid. A newly generated invoice remains unpaid until an admin performs the payment action.
5

Mark an invoice paid or validate it

Payment and validation are deliberate admin actions.

Method A — Customer Records

  1. Open Customer Records.
  2. Select the customer.
  3. Scroll to Customer Invoices.
  4. Choose Mark paid or Validate.

Method B — Invoice Lookup

  1. Open Invoice Lookup.
  2. Enter the complete invoice number.
  3. Choose Find invoice.
  4. Use Mark paid, Mark unpaid, Validate or Public view.
Generated: Unpaid→Admin confirms payment→Paid
PDF download ≠ payment. Only the Mark paid button changes the payment status. Mark unpaid can reverse an incorrect payment entry.

7

Set prices, stock and order rules

These settings control customer totals and daily workflow.

Sidebar→Settings
₨

Fixed Bottle Prices

Enter PKR per unit for each bottle type, then choose Save Bottle Prices.

📦

Live Bottle Inventory

Enter available units and choose Save Inventory.

⚙️

Order Workflow

Configure auto-accept, notifications, delivery confirmation, cancellation and cutoff time.

💾

Save each section

Each settings card has its own save button; complete it before leaving.

If a customer sees a zero price, return to Settings → Fixed Bottle Prices, enter every required price and save again.
8

Manage users and passwords

Owner-controlled account administration.

Sidebar→All Users

Create an admin

  1. Complete New Admin.
  2. Enter name, email and password.
  3. Select Admin, Manager or Owner.
  4. Choose Create Admin.

Reset a customer password

  1. Find the user under All Customer Users.
  2. Choose Reset password.
  3. Enter or generate a temporary password.
  4. Enter the owner password and choose Set password.
Share temporary passwords privately. Removing an admin requires the owner password; removing a customer requires typing DELETE.
